Vamos testar algumas formatações de texto

Para usarmos o texto em negrito usamos a tag <strong>
Para usarmos o texto em itálico usamos a tag <em>
Para marcarmos um texto usamos a tag <mark>
Para deixar um texto menor usamos a tag <small>
Para deixar o texto riscado, indicando que deve ser desconsiderado usamos a tag <del>
Para deixar o texto sublinhado, indicando que deve ser lido com atenção usamos a tag <ins>

H²O Usamos a tag <sub>
X 2 Usamos a tag <sup>

Para escrevermos um código de programação usamos duas tag uma dentro da outra <pre> e <code>, a primeira para que o código seja ostrado como foi digitado, com todas as quebras de linhas e a segunda fique mono-espaçado.

Para fazer citações, usamos a tag <blockquote>. Podemos usar o parâmetro cite="" para colocar o link para a citação.
Exemplo:

Gênesis 1:1: "No princípio Deus criou os céus e a terra."


Podemos usar abreviações com a seguinte tag <abbr>
Exemplo:ATCJ
Podemos usar como parâmetro title="" para quando passar o mouse na abreviação mostrar o significado dela.
